# NotebookLM Experiment Log

## Tool Evaluated
Google NotebookLM

## Scenario
Evaluation of NotebookLM as an AI-powered educational assistant for university students.

## Source Document
Assessment report containing:
- ESG chatbot implementation
- Snowflake cloud security breach analysis
- LLM ESG message triage system

## Experiments Conducted

### Experiment 1 – Summary Generation
Prompt:
"Summarise the key topics covered in this report."

Result:
NotebookLM accurately summarised the major themes and identified cloud security, AI and ESG concepts.

### Experiment 2 – Question Answering
Prompt:
"What are the key cloud security lessons from the Snowflake breach analysis?"

Result:
NotebookLM correctly identified MFA, IAM and the Shared Responsibility Model.

### Experiment 3 – Study Guide
Result:
NotebookLM generated revision questions and study materials based on the uploaded report.

### Experiment 4 – Audio Overview
Result:
NotebookLM converted report content into an audio learning resource.
